# MiSTer Updater script for PC (Windows and Linux)
Download [*updater-pc.zip*](https://github.com/theypsilon/Update_All_MiSTer/blob/master/updater-pc/updater-pc.zip?raw=true) and extract *updater-pc* directory on the root of your MiSTer SD (don't put it in any SD subdirectory, *updater-pc* directory must reside in its root). On Windows execute *update-win.bat*, on Linux execute *update-linux.sh*.

Any configuration file should be placed at *updater-pc/update_all.ini* (optional)
